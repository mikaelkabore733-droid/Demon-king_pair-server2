Guide

Comment générer ta session

Suis ces étapes simples pour connecter ton bot WhatsApp à DEMON KING GEN.

01
Ouvre le Générateur

Clique sur "Générateur" dans la navigation ou le bouton principal de la page d'accueil. Choisir entre Pair Code ou QR Code selon ta préférence.

02
Entre ton numéro WhatsApp

Saisis ton numéro complet avec l'indicatif pays (ex: +225 pour Côte d'Ivoire). Ne mets pas d'espaces ni de tirets.

+2260123456789
03
Clique sur Générer

Appuie sur le bouton GÉNÉRER LE CODE. Attends quelques secondes — ton code apparaît dans la zone rouge.

04
Copie et colle dans WhatsApp

Pour le Pair Code : ouvre WhatsApp → Appareils liés → Lier un appareil → Saisir le code à 8 chiffres. Pour le QR : scanne directement avec la caméra WhatsApp.

05
Ton bot est connecté 🜲

C'est tout ! La session est générée. Copie l'ID de session et colle-le dans la configuration de ton bot. Le Démon Roi approuve.
